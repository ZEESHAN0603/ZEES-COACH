✍️ ZEES-COACH – AI-Powered Personalized Language Learning Platform

An AI-powered personalized language learning assistant developed during the AI Ignite National GenAI Hackathon, designed to help children—especially NRI students—learn their native languages such as Tamil and Telugu through handwriting recognition, conversational AI, and cultural learning.

The platform acts as a 24/7 AI tutor, capable of reading handwritten notes, correcting grammar, and guiding students with intelligent feedback.

🚀 Project Overview

Millions of Indian children living abroad lack access to personalized native language tutors. ZEES-COACH solves this by using OCR, Generative AI, and Large Language Models to provide automated handwriting correction and AI-based tutoring.

The system enables students to:

📷 Scan handwritten notebook pages
🧠 Extract and identify text using OCR and AI
✍️ Correct spelling and grammar automatically
🤖 Receive personalized tutor-style feedback
🌍 Learn native languages anytime, anywhere

🎯 Problem Statement

AI-Powered Personalized Learning Platform

Over 50 million Indian children living abroad struggle to learn their native languages due to lack of personalized tutors.

Expected Solution Components:

• AI Handwriting Tutor – Extract and correct handwritten text using OCR
• Voice / AI Tutor – Provide conversational learning support
• Cultural Learning Module – Deliver educational and cultural content

ZEES-COACH implements the AI Handwriting Tutor and AI Tutor modules using advanced AI models.

⚙️ How It Works

Student writes in a physical notebook
Image is captured using mobile or uploaded
Tesseract OCR extracts handwritten text
Gemini AI refines and identifies the text
Llama 3.2B corrects grammar and provides feedback
Corrected text is shown to the student
